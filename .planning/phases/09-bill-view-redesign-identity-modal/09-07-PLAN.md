<objective>
Close GAP-09-FLOW: unify the main create flow on the collaborative `/split/[sessionId]` surface so the bill creator actually reaches the Phase 9 work (identity modal + collaborative Bill View) in normal single-device use.

Output: A shared `lib/createSession.ts` helper, a rewired SetupStep Continue handler, a deduplicated ShareLinkButton, an app/page.tsx that no longer routes through the old wizard and resumes into /split when a session already exists, and a test proving Continue → session create → /split navigation.
</objective>
