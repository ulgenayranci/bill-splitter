## Phase Goal

**As a** dinner host, **I want to** share a link so each person claims their own items on their own phone, **so that** I don't have to tap through everyone else's orders myself.

This is Wave 0 of three plans delivering the user story. Wave 0 lays the foundation — Wave 1 ships the server slice (so a POST /api/session call returns a sessionId), Wave 2 ships the client slice (so the host actually sees and shares a URL and guests claim items).
